# Showcase — overhauled buildings, running, per tier

Generated by [`tools/showcase.py`](../tools/showcase.py). Each building is
composited exactly the way the game loader does (`lib/sprite.lua` +
`lib/reskin.lua`): **base → mask × tier-tint → emission (additive glow)**, with
every layer aligned by its own `shift`. Tier tints (yellow / red / blue) come
straight from [`lib/tiers.lua`](../lib/tiers.lua).

## Files

Four composite grids — one row per building, columns = tiers 1→3 left to right
(the nanofabricator line only has tiers 1–2):

| File | Buildings |
|---|---|
| `group-1-electrolyzer-compressor.gif` | Electrolyzer, Compressor |
| `group-2-airfilter-crusher-foundry.gif` | Air filter, Crusher, Foundry |
| `group-3-vacuum-flotation-nanofabricator.gif` | Vacuum chamber, Flotation cell, Nanofabricator |
| `group-4-geothermal-hydro.gif` | Geothermal plant, Hydro plant |

Plus one running GIF per building per tier under `<source>/<source>-tier<N>.gif`.

## Regenerate

```
python3 tools/showcase.py                 # all buildings + the four grids
python3 tools/showcase.py --only oxidizer  # one building
```

Useful flags: `--cell` (grid cell px), `--fps`, `--max-frames` (subsample to
keep GIFs small; `0` = every frame), `--bg` (background shade), `--no-singles`.

## Notes / limitations

- Shows the **"after"** (overhauled) art only — no before/after comparison yet.
  The original Nullius art is assembled at load time from Angel's graphics with
  per-tier Lua tints, so a faithful "before" needs in-game capture.
- The **drop shadow** and the **recipe-tint (color2)** layer are omitted
  (cosmetic / need runtime recipe context).
- Emission is shown **always-lit** so the building reads as running; in-game the
  glow is only lit while crafting.
- GIF is heavy for this much animated detail (~3–6 MB per grid at defaults). For
  much smaller files, render an MP4/WebM from the frames instead.
